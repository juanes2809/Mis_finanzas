# 💸 MisFinanzas

App personal de finanzas con estadísticas semanales, categorías y metas de ahorro.

## 🚀 Cómo subir a Vercel (GRATIS, en 5 minutos)

### Opción A — Sin instalar nada (recomendada)

1. Ve a **https://vercel.com** → crea cuenta gratis con Google/GitHub
2. En el dashboard haz clic en **"Add New Project"**
3. Selecciona **"Import from Git"** → primero sube este proyecto a GitHub:
   - Ve a **https://github.com/new** → crea repositorio (puede ser privado)
   - Sube los archivos o usa GitHub Desktop
4. En Vercel selecciona tu repo → clic en **Deploy**
5. ¡Listo! En 2 minutos tendrás una URL como `https://misfinanzas.vercel.app`

### Opción B — Con Node.js instalado

```bash
# Instala dependencias
npm install

# Prueba localmente
npm run dev

# Construye para producción
npm run build

# Instala Vercel CLI y despliega
npm install -g vercel
vercel --prod
```

## 📱 Agregar al celular como app

### Android (Chrome)
1. Abre tu URL en Chrome
2. Menú (⋮) → "Agregar a pantalla de inicio"

### iPhone (Safari)
1. Abre tu URL en Safari
2. Botón compartir → "Agregar a pantalla de inicio"

¡Quedará como ícono igual que una app nativa! Los datos se guardan en el celular.
